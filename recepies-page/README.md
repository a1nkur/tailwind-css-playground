# Commands

## Generate - tailwind.config.js

- Install tailwindcss

```bash
    $ npm install tailwindcss

```

## Generate - tailwind.config.js

- Do not directly modify this file.

```bash
    $ npx tailwindcss init --full

```

## Generate new tailwind.config.js

- Extends default configuration

```bash
    $ npx tailwindcss init

```
